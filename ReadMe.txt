->開發平台:
(1) Microsoft Window 9x OS
(2) Microsoft Visual C++ v6.0

->使用說明:
(1) 確認 sic.asm 檔案與sic.exe 同在一個資料夾內
(2) 打開MS-DOS作業視窗
(3) 找到sic.exe所在目錄
(4) 鍵入sic開始執行組譯程式
(5) 當程式詢問輸入組合語言原始碼時,鍵入sic.asm
(6) 程式執行中,會產生報表檔(sic.list)及目地檔(sic.obj),並且在螢幕上會顯示Object Code.
(7) 報表檔(sic.list)及目地檔(sic.obj)可使用"記事本"(應用軟體)觀看
